# Sistema de Dirección IA

La IA no empieza en el prompt. Empieza en la dirección.

Este repositorio contiene la estructura en crudo del **Sistema de Dirección IA**, un andamiaje de contexto diseñado para interrumpir la convergencia hacia el promedio estadístico en modelos generativos (Claude, VEO 3, ElevenLabs).

## Sede Técnica y Documentación Completa
La evaluación metodológica de este sistema y el desarrollo completo de su arquitectura ("Documento Cero") residen en la sede digital oficial: 
🔗 **[danicollada.es](https://danicollada.es)**

## Arquitectura del Sistema
El sistema se compone de 9 módulos de información no opcionales. Cuando faltan, el modelo rellena los huecos con probabilidad estadística global.

1. **Intención**: Efecto cognitivo objetivo.
2. **Contexto**: Capas de realidad del proyecto.
3. **Identidad**: Invariantes de marca.
4. **Referencias**: Lógica externa asimilada, no copiada.
5. **Restricciones**: Identidad en negativo (Negative Constraints).
6. **Producción**: Parámetros de ejecución.
7. **Auditoría**: Criterios de evaluación del output.
8. **Publicación**: Sistema de distribución.
9. **Aprendizaje**: Bucle de retroalimentación.

## Componentes Operativos
En el directorio `/arquitectura` se incluyen los esquemas JSON y Markdown utilizados en producción, incluyendo el `continuity_lock.json` para modelos de generación de vídeo.
